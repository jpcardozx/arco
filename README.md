# ARCO

Modern React application with clean architecture.

## Structure

```
src/
├── app/            # Next.js app directory
├── components/     # UI components (4 categories)
├── design-system/  # Design system
├── lib/           # Utilities
├── types/         # TypeScript types
└── styles/        # Global styles
```

## Development

```bash
npm run dev
npm run build
npm run test
```
